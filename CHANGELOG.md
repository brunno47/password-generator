# Changelog

Todas as mudanças notáveis neste projeto serão documentadas neste arquivo.

O formato é baseado em [Keep a Changelog](https://keepachangelog.com/pt-BR/1.0.0/),
e este projeto adere ao [Semantic Versioning](https://semver.org/lang/pt-BR/).

## [1.0.0] - 2024-01-XX

### Adicionado
- Sistema completo de autenticação (registro e login)
- Geração de senhas com 3 modos: Normal, Memorável e PIN
- Opções personalizáveis de caracteres
- Análise de força de senha em tempo real
- Cálculo de entropia e tempo estimado para quebrar senha
- Histórico de senhas por usuário
- Exportação em TXT, JSON e CSV
- Importação de histórico
- Interface moderna com design glassmorphism
- Animações e efeitos visuais
- Responsividade completa para mobile
- Card de perfil do usuário
- Sistema de notificações
- Análise detalhada de padrões de senha
- Sugestões de melhoria de segurança
- Geração de múltiplas senhas
- Exclusão de caracteres similares
- Modo memorável com palavras em português

### Segurança
- Geração criptograficamente segura usando Web Crypto API
- Armazenamento isolado por usuário
- Sessões persistentes

### Interface
- Design moderno e elegante
- Animações suaves
- Feedback visual em todas as ações
- Indicadores de força de senha
- Estatísticas em tempo real
